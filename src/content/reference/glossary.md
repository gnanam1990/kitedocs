# Glossary

- **AgentID** — the Kite-native identity record for an autonomous agent (or human acting like one). See [agentid-seven.vercel.app](https://agentid-seven.vercel.app).
- **kpass** — Kite's session-token system for browsing the agentic web. A "kpass session" is a signed bundle authorizing scoped actions.
- **KITE** — the native token of the Kite chain. 18 decimals. Used for gas.
- **KiteAuth** — community sign-in SDK ([repo](https://github.com/gnanam1990/kiteauth)).
- **KiteScan** — block explorer for Kite. Mainnet at [kitescan.ai](https://kitescan.ai).
- **Test USDT** — testnet ERC-20 stand-in for USDC. Address `0x0fF5…7e63`, **18 decimals** (not 6).
- **USDC.e** — bridged USDC on Kite Mainnet (planned). Treat as the canonical stablecoin once published.
- **x402** — the HTTP 402 Payment Required pattern, used by agents to request payment mid-request.
