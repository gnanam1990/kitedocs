# FAQ

### Is Kite EVM-compatible?

Yes. Standard JSON-RPC, standard tools (viem, ethers, web3.py), standard ERC-20s.

### Where do I get testnet KITE?

Use [KiteFaucet](https://github.com/gnanam1990/kitefaucet) — 5 KITE + 10 Test USDT per 24h.

### What's the canonical stablecoin?

On Testnet: Test USDT (`0x0fF5…7e63`, 18 decimals).
On Mainnet: USDC.e once published. Check [docs.gokite.ai](https://docs.gokite.ai) for the live address.

### How is Kite different from Ethereum L2s?

Kite is its own chain, not an L2. It optimizes for agent-to-agent micro-payments and stablecoin throughput.

### Can I bridge from Ethereum?

Bridge UI is in PREVIEW — see [KiteBridge UI](https://github.com/gnanam1990/kitebridge-ui) when available.

### Are these docs official?

No. KiteDocs is community-maintained. The official source is [docs.gokite.ai](https://docs.gokite.ai).
